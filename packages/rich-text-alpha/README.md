# @contentful/field-editor-rich-text-alpha

Disclaimer ⚠️:

- This package contains experimental code for the rich-text editor.
- The code is not production-grade and Contentful doesn't provide any form of support for it.
- The package may be removed in future without notice.
- This package is not available on npm.

## Installation

```sh
npm install @contentful/field-editor-rich-text-alpha \
    react@^19.1.0 \
    react-dom@^19.1.0 \
    react-reconciler@0.32.0
```

_Note_: `@contentful/field-editor-rich-text-alpha` does not require a specific version of React. **However**, you must ensure that your React Reconciler version matches your React/React DOM versions.

| React version   | React Reconciler version |
| --------------- | ------------------------ |
| 19.x            | 0.32.0                   |
| >= 18.2.0, < 19 | 0.29.0                   |
| 18.1.x          | 0.28.0                   |
| 18.0.x          | 0.27.0                   |
| 17.x            | 0.26.1                   |
